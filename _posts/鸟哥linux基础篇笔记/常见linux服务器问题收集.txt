笔记本装centos7，合上盖子后笔记本休眠问题：
/etc/systemd/login.conf ，handlelidswitch=suspend参数改成ignore或lock，删掉前面的#，保存后运行命令systemctl restart systemd-logind使之生效；

安装套件后出现 libiw.so.29(缺少的动态链接库):can not open shared object file:no such file or directory
原因：缺少动态链接库
解决：修改/etc/ld.so.conf，在该文件中加入安装套件的动态数据库位置如/usr/local/lib，执行ldconfig命令。

kali2.0修改ip：
/etc/networking/interfaces 文件中增加如下行：
auto eth0
address x.x.x.x
netmask x.x.x.x
gateway x.x.x.x

安装rz、sz工具：
yum install lrzsz 

查看系统版本：
(1)lsb_release -a
(2)uname、uname -r、uname -a
这个命令适用于所有的linux，包括Redhat、SuSE、Debian、Centos等发行版。
(3)cat /etc/redhat-release 
   rpm -q redhat-release
   rpm -q centos-release
这两种方法适用于RedHat,CentOS
(4)cat /proc/version
当前centos、redhat对应版本的命令。

centos安装mysql数据库：
yum install mariadb-server  mariadb是mysql的社区版，默认监听3306端口；
service mysql start 启动mariadb数据库。

NIC 网卡 
ISP 网络运营商
AP 无线接入点 wireless access point
ESSID  AP名称
ifup/ifdown 是脚本，/etc/sysconfig/network-scripts 下存在ifcfg-ethx 才能对ethx 开启关闭。 

dmesg | grep -in eth   查看网卡是否被系统抓到及网卡使用的驱动模组（如e1000）。结合dmesg | cat -n | less 搜寻行号查看网卡详细信息
lspci | grep -i ethernet 查看网卡信息。lspci 用来查看所有连接到系统pci总线上的设备。
lsmod | grep e1000，查看核心是否载入该驱动程序e1000
modinfo e1000 查看驱动程序e1000的详细信息

lsusb 对于外接的usb无线网卡，使用该命令查看系统是否捉到网卡
iwconfig 查看被捉到的外接无线网卡驱动程序是否被加载，出现 wlan0之类的网卡名称 说明网卡驱动被加载成功

更新网卡驱动步骤：
下载驱动程序tarball，编译安装，新的网卡驱动安装在/lib/modules/`uname -r`/kernel/drivers/net/xx/xx.ko；
rmmod e1000 ，卸载已经加载的网卡驱动(如e1000)；
modprobe e1000 加载更新后的网卡驱动程序；
modinfo e1000 查看更新后的网卡驱动程序e1000信息；
vi /etc/modprobe.d/ether.conf 在/etc/modprobe.d下新建文件ether.conf并增加行alias ethx e1000 (多网卡每个ethx占一行) 设定开机启动网卡驱动


/etc/modprobe.conf 无线网卡装置模块与代号对应关系设定文档，修改后reboot生效
/etc/sysconfig/network 设置主机名HOSTNAME与是否可以连接因特网
/etc/sysconfig/network-scripts/ifcfg-ethx 设置网卡参数
/etc/resolv.conf 域名解析服务器IP地址设定文件
本机连接到该文件下指定的DNS服务器上查找指定IP对应的主机名称。
/etc/hosts 设定主机名称与IP地址对应关系
连接FTP/POP3服务器延迟30-60秒，是因为主机默认开启反查主机IP地址对应主机名称机制，内网主机IP地址对应的主机名称
不能连接internet通过/etc/resolv.conf中设定的DNS服务器查找IP对应主机名称，导致验证不能通过，延迟默认的30-60s
/etc/services 各协议使用的端口号设定文件

无线网卡
ifconfig ethx up/down ，同ifup/ifdown ethx，启动网卡；
iwconfig ethx 查看无线网卡信息；
iwconfig ethx 设定无线网卡连接到无线ap的参数：
iwlist ethx scanning 搜寻附近AP，
iwconfig ethx essid "AP名称（essid）"  key "。。。"  连接AP；
dhclient ethx 通过dhcp服务器获取ethx动态ip地址；
/etc/rc.d/rc.local 下增加行 /sbin/ifconfig ethx up 设定无线网卡开机启动。

网卡必须设定的参数：
device=ethx，必须与ifcfg-ethx 对应
hwaddr= ，mac地址，只有一张网卡时可省略，如果有两张一模一样的网卡且使用相同的驱动程序，则需要表明该网卡mac地址
onboot=yes，是否预设启动此设定档文件
bootproto=dhcp/static 
ipaddr= ，static时的网卡ip地址
netmask= ，网卡子网掩码
gateway= ，默认网关，有多张网卡时，只在一张网卡上设定即可，避免重复设定
nv_controlled=no，是否允许其他软件管理网卡

dig www.baidu.com 验证是否正确访问dns服务器，能返回域名对应的ip即设定正确。

/etc/sysconfig/network、/etc/hosts 主机名称编辑文档，hostname查看本机名称。

修改连接到ap的设定参数：
ifconfig wlan0 down && rmmod e1000，关闭wlan0并卸载无线网卡驱动程序；
vi /etc/wireless/e1000/e1000.dat ，修改连接ap的设定参数；
modprobe e1000 &&ifconfig wlan0 up ，加载无线网卡驱动程序并启动无线网卡

centos7给网卡设定静态ip地址步骤：
注释掉bootproto=dhcp参数；
onboot=yes ；
增加静态ip地址参数ipaddr0、prefix0=24、gateway0、dns ；
重启网卡，配置生效。


centos7连接无线wifi上网配置：
dmesg | grep -i firmware 查看无线芯片是否需要固件支持，若无输出表明无线芯片不需要系统固件支持；
iw dev 查看无线网卡信息，若系统成功加载无线芯片，可以看到无线芯片信息；
ip link set wlp3s0 up 启动无线网卡wlp3s0；
ip link show wlp3s0 查看wlp3s0是否启动；
iw wlp3s0 link 查看无线网卡连接情况；
iw wlp3s0 scan | grep -i essid  搜寻无线wifi信号；
wpa_supplicant -B -i wlp3s0 -c  <(wpa_passphrase "essid名称" "psk即密码") ，连接无线ap。
dhclient wlp3s0 通过dhcp服务器获取无线网卡ip；
ip addr show wlp3s0 查看无线网卡ip地址信息；
iw wlp3s0 link 再次查看无线网卡wlp3s0网络连接情况。

wpa_supplicant 命令：-B 表示在后台执行，-i 指定网卡接口，-c 指定验证信息。

ip link set ethx name xxx 将网卡ethx改名为xxx；

ifconfig eth0:0 192.168.1.111 在eth0上虚拟一个网卡eth0，并给他分配ip地址192.168.1.111

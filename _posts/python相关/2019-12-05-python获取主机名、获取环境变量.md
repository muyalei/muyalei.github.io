#获取环境变量

```
import os

env_dist = os.environ # environ是在os.py中定义的一个dict environ = {}

print env_dist.get('JAVA_HOME')
print env_dist['JAVA_HOME']

# 打印所有环境变量，遍历字典
for key in env_dist:
    print key + ' : ' + env_dist[key]
```

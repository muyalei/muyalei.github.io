/etc/dhcpd.conf ，dhcp服务设定文件，若安装dhcp套件后不存在该文件，新建该文件，复制/etc/dhcpd.conf.sample内容到该文件下，进行编辑
该设定文件中，每一行以分号；结尾，#代表批注
服务全局参数配置
vi /etc/dhcpd.conf
default-lease-time xxx；，设定租约期限，单位是 秒
max-lease-time xxxx； ，恕约期限，可以超过租约期限但不能超过该时限设定
option domain-name  "dhcp.vbird.org"； 设定被分配IP地址的主机域名
option domain-name-servers a.a.a.a，b.b.b.b ；设定dns服务器，多个dns服务器用逗号 ，分隔
ddns-update-style interim；
设定动态IP分配
subnet 192.168.1.0 netmask 255.255.255.0 {
       range 192.168.1.21 192.168.1.100；
       option broadcast-address 192.168.1.255；
       option routers 192.168.1.2；    
}
设定静态IP分配
host muyalei {
     hardware ethernet mac地址；
     fixed-address 192.168.1.33；
     option broadcast-address 192.168.1.255；
     option routers 192.168.1.2；
}
touch /etc/dhcpd.leases ，建立租约期限文件，跟踪联机时间
/var/lib/dhcp/dhcpd.leases ，dhcp服务启动后，手工建立的/etc/dhcpd.leases文件被复制为该文件，跟踪联机时间
编辑启动脚本
vi /etc/rc.d/init.d/dhcpd
CONFIGFILE="/etc/dhcpd.conf" ，dhcp服务启动时，配置参数加载路径
LEASEFILE="/var/lib/dhcp/dhcpd.leases" ，租约期限记录文件路径
INTERFACES="eth0 eth1" ，对eth0、eth1两个接口开放dhcp服务
OPTIONS="-q"  ，我也不知道这是在设定啥
start(){
       daemon /usr/sbin/dhcpd -cf $CONFIGFILE -lf $LEASEFILE $OPTIONS $INTERFACES
}
/var/log/messages ,查看登陆信息确认dhcp服务是否启动成功
netstat -tunlp ，查看dhcp服务是否已经在监听端口。dhcp协议由bootps协议演化来。
/etc/hosts，在该文件下添加所有需要dhcp服务的主机ip地址与主机名的对应关系
